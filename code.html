Main Project

Sona waveform processing platform

My Github repository, which introduces basic models in Python and PyQT5. The purpose of this project is to create a software for audio noise reduction for sonar detection.
  • In order to achieve noise reduction, it is necessary to collect public noise data sets and add noise to the pure audio detected by sonar.
  • Use Scipy to design a sonar audio conversion batch processing module, which is used to convert the original sonar audio Pcm format into Convert to Wav format to facilitate data preprocessing and model training.
  • The Wavenet network is used for noise reduction in sonar seepage sampling, and the effect is better. Using GBDT2nn Realize waveform classification and classify sonar waveforms into those with seepage and without seepage.
  • Use PYQT5 to support the front-end design of the waveform analysis platform, which mainly includes time domain and frequency domain waveform analysis and Processing module, percolation noise reduction module, Fourier noise reduction, spectral subtraction noise reduction, Wavenet noise reduction and other denoising module.

Sumo-RL

My Github repository, AI competition in traffic. The purpose of this project is to control traffic light.
  • Multi-agent, each phase is defined as an agent. The agent predicts the pressure value and expected release time of the current phase, and releases the one with the largest Q value.
  • The road blocking mechanism predicts the current problematic road conditions (the prediction is based on the distance between the first car and the intersection, etc.), and puts the current road into a blacklist, and waits for it to return to normal.
  • Make a feature processing module similar to dual GRU to generate a state knowledge memory (TODO).

Strengthen the foundation

My Github repository, War game confrontation project.
  • Mainly responsible for building reinforcement learning models including QMIX, MADDPG, MAPPO and other algorithms, encapsulating the environment interface design for agent interaction into Gym form, and designing the Redis situation information reading interface.
  • Use LSTM and Attention mechanisms to build a situation processing module, including situation prediction and feature extraction. The extracted features are input to the agent model, and the action sequence is generated through the decision-making layer.
  • Use the B3tree behavioral tree engine to build a rule agent and create a rule data set for adversarial training in reinforcement learning.
